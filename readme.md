# PortfolioHUB

Portfólio central de projetos acadêmicos e pessoais de Lucas Cortes Teixeira.

---

# Objetivo

Este repositório foi criado para centralizar e apresentar projetos
desenvolvidos durante estudos, desafios acadêmicos e projetos pessoais.

O objetivo é organizar o portfólio de maneira profissional,
permitindo fácil navegação entre os repositórios individuais.
