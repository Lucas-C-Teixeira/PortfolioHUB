## Sobre o Projeto

O **PortfolioHUB** é um repositório central criado para reunir, organizar e apresentar projetos acadêmicos e pessoais desenvolvidos por **Lucas Cortes Teixeira**.

O objetivo deste projeto é construir uma vitrine profissional de desenvolvimento de software, concentrando aplicações, estudos e documentações em um único ambiente acessível via GitHub Pages.

Este repositório funciona como um portal central para navegação entre projetos independentes hospedados no GitHub.

---

# Objetivos

- Centralizar projetos acadêmicos e pessoais
- Organizar documentação técnica
- Demonstrar evolução prática em desenvolvimento de software
- Construir presença profissional no GitHub
- Disponibilizar um portfólio online responsivo
